## ğŸš€ Quick Start

### Installation

**Option 1: Clone the repository**
```bash
git clone https://github.com/Vishal710-max/profile-card.git
cd profile-card
```

**Option 2: Download ZIP**
Download the latest release and extract the files.

**Option 3: Copy the code**
Simply copy the HTML/CSS/JS from the example file into your project.

### Basic Usage

1. **Include the files in your HTML:**
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="profile-card.css">
    <title>My Profile</title>
</head>
<body>
    <div class="profile-card">
        <!-- Profile card content -->
    </div>
    <script src="profile-card.js"></script>
</body>
</html>
```

2. **Customize your details:**
Update the avatar, name, bio, and social links in the HTML.

3. **Adjust styling (optional):**
Modify CSS variables in `profile-card.css` to match your brand colors.

## ğŸ¨ Customization

### CSS Variables

Easily customize the appearance by modifying these CSS variables:

```css
:root {
    --primary-color: #6366f1;
    --background-color: #ffffff;
    --text-color: #1f2937;
    --card-shadow: 0 10px 30px rgba(0, 0, 0, 0.1);
    --border-radius: 16px;
    --font-family: 'Inter', -apple-system, sans-serif;
}
```

### Adding Social Links

```html
<div class="social-links">
    <a href="https://github.com/yourprofile" aria-label="GitHub">
        <i class="icon-github"></i>
    </a>
    <a href="https://linkedin.com/in/yourprofile" aria-label="LinkedIn">
        <i class="icon-linkedin"></i>

    <!-- Add more social links -->

```
